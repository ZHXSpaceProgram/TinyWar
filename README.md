# Tiny War: 2D Pixel Turn-Based Strategy Game on Python

## How to Play?

If you are on Windows:
- Download the latest [release](https://github.com/ZHXSpaceProgram/TinyWar/releases)
- Unzip
- Run the excutable file

To run source code:


```
pip install pygame
python src/main.py
```
