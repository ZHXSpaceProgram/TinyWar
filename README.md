## How to Play?

If you are on Windows:
- Download `Game.zip` from the latest [release](https://github.com/ZHXSpaceProgram/TinyWar/releases)
- Unzip
- Run the excutable file

To run source code:

```
pip install pygame
python src/main.py
```
